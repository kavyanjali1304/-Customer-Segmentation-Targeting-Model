# Customer-Segmentation-Targeting-Model
What's Inside
5 Customer Segments (K-Means style):

👑 Champions — high spend, frequent buyers
💎 Loyalists — consistent, moderate spend
🌱 Potentials — growing, low-mid spend
⚠️ At Risk — previously active, dropping off
😴 Hibernating — long inactive

Interactive Features:

🔵 Scatter plot — 500 customers plotted by Spend vs Frequency, hover to inspect any dot
🔍 Segment filter tabs — click any segment, entire dashboard filters instantly
📊 RFM score bars — Recency, Frequency, Monetary per segment
🎯 Targeting strategies — campaign recommendations with expected ROI
⚡ Live classifier — type any customer's data → auto-assigned to a segment
