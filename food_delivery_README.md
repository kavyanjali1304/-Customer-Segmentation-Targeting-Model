# 🍔 QuickBite — Food Delivery Business Analytics Dashboard

An interactive **Business Intelligence Dashboard** for a food delivery platform, built with pure HTML, CSS, and JavaScript.

> 🎯 Resume project demonstrating Business Development Associate skills: market analysis, revenue tracking, partner management, and growth insights.

---

## 🚀 Live Demo

Host on **GitHub Pages**:
1. Repo → **Settings** → **Pages** → Source: `main` / `root`
2. Live at: `https://<your-username>.github.io/food-delivery-bda/`

Or just open `index.html` in any browser — works fully offline.

---

## 📊 Dashboard Features

| Section | What It Shows |
|---|---|
| 📈 KPI Cards | Revenue, Orders, Partners, Avg Rating with QoQ change |
| 💰 Revenue Chart | Monthly Actual vs Target (₹ Lakhs) |
| 🗺️ Top Cities | Revenue share by city with visual bars |
| 🍽️ Category Mix | Cuisine breakdown donut chart |
| 🤝 Top Partners | Restaurant-wise revenue & growth |
| 📦 Order Flow | End-to-end fulfilment pipeline |
| 🔥 Order Heatmap | Demand by day & hour of week |
| 💡 Growth Insights | BDA-style recommendations for expansion |

---

## 🛠️ Tech Stack

- **HTML5 / CSS3 / JavaScript** — No frameworks, no dependencies
- **Google Fonts** — Playfair Display + Outfit
- **SVG** — Custom donut chart rendered in-browser
- **CSS Grid & Flexbox** — Fully responsive layout
